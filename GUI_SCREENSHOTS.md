## 🎯 주요 버튼 설명

### 🔍 찾아보기 버튼
- **단일 폴더**: L3A 폴더 하나를 선택
- **배치 처리**: 여러 L3A 폴더가 있는 상위 폴더 선택

### 🚀 처리 시작 버튼
- 선택한 폴더의 처리 시작
- 클릭 시 확인 대화상자 표시 (배치 처리)
- 처리 중에는 비활성화됨

### ⏹️ 중지 버튼
- 처리 중에만 활성화
- 클릭 시 처리 중단 확인
- 안전하게 중지됨

### 📂 출력 폴더 열기 버튼
- ProcessedOutputs 폴더를 파일 탐색기로 열기
- 결과 파일 빠르게 확인

## 📋 사용 시나리오

### 시나리오 1: 단일 폴더 처리

```
1. "단일 폴더" 옆 [찾아보기] 클릭
2. L3A_202401120060_20230316_37709024 선택
3. [🚀 처리 시작] 클릭
4. 진행 상황 확인
5. 완료 메시지 확인
6. [📂 출력 폴더 열기] 클릭하여 결과 확인
```

**예상 소요 시간:** 약 2-3분

---

### 시나리오 2: 배치 처리 (5개 폴더)

```
1. "배치 처리" 옆 [찾아보기] 클릭
2. D:\국토정보위성 선택
3. 로그에서 발견된 폴더 확인
   → [13:27:10] 발견된 L3A 폴더: 5개
4. [🚀 처리 시작] 클릭
5. 확인 대화상자에서 [예] 클릭
6. 진행 상황 확인 (1/5, 2/5, ...)
7. 완료 메시지 확인
   → 전체: 5개 | 성공: 5개 | 실패: 0개
```

**예상 소요 시간:** 약 10-15분 (폴더당 2-3분)

## 💡 실용 팁

### 빠른 작업
1. **자주 사용하는 폴더 저장**: 경로를 복사해두기
2. **옵션 미리 설정**: 필요한 결과만 생성
3. **배치 처리 활용**: 여러 날짜를 한번에

### 문제 해결
1. **로그 확인**: 오류 메시지를 정확히 읽기
2. **폴더 구조 확인**: L3A로 시작하는지 확인
3. **재시도**: 네트워크/권한 문제 시 재실행
