## 📖 사용 단계

### 1️⃣ 폴더 선택

**단일 폴더 처리:**
1. "단일 폴더" 옆의 `찾아보기` 버튼 클릭
2. L3A 폴더 선택 (예: `L3A_202401120060_20230316_37709024`)
3. 확인 클릭

**배치 처리 (여러 폴더):**
1. "배치 처리" 옆의 `찾아보기` 버튼 클릭
2. L3A 폴더들이 있는 상위 폴더 선택 (예: `D:\국토정보위성`)
3. 확인 클릭
4. 로그에 발견된 폴더 목록이 표시됨

💡 **Tip**: 둘 다 선택하면 모든 폴더가 처리됩니다!

---

### 2️⃣ 옵션 설정

**✅ RGB 합성 이미지 생성**
- 체크: Blue, Green, Red 밴드를 합쳐서 컬러 이미지 생성
- 권장: 체크 ✓

**✅ NDVI 계산**
- 체크: 식생 지수 계산
- 권장: 체크 ✓

**☐ 처리 후 QGIS 열기**
- 체크: 처리 완료 후 QGIS 프로젝트 자동 실행
- 권장: 필요시 체크

---

### 3️⃣ 처리 시작

1. `🚀 처리 시작` 버튼 클릭
2. (배치 처리의 경우) 확인 대화상자에서 `예` 클릭
3. 진행 상황을 진행 바와 로그에서 확인

**처리 중:**
- 진행 바가 움직임
- 로그에 실시간 정보 표시
- 상태 레이블에 현재 작업 표시

---

### 4️⃣ 결과 확인

**처리 완료 후:**
1. 완료 메시지 박스 확인
2. `📂 출력 폴더 열기` 버튼 클릭
3. ProcessedOutputs 폴더에서 결과 파일 확인

**생성된 파일:**
- `*_B_georef.tif` - Blue 밴드
- `*_G_georef.tif` - Green 밴드
- `*_R_georef.tif` - Red 밴드
- `*_N_georef.tif` - NIR 밴드
- `RGB_Composite_*.tif` - RGB 합성
- `NDVI_*.tif` - NDVI 결과
- `kompsat_ndvi_project.qgz` - QGIS 프로젝트

## ⚡ 주요 기능

### 🎯 단일/배치 처리
- **단일**: 한 개 폴더만 처리
- **배치**: 여러 폴더를 한번에 자동 처리

### 📊 실시간 진행 상황
- 진행 바로 전체 진행률 표시
- 로그 창에서 상세 정보 확인
- 현재 처리 중인 파일명 표시

### ⏹️ 중지 기능
- 처리 중 언제든 중지 가능
- 중간까지 처리된 파일은 저장됨

### 🎨 색상 코딩 로그
- 🔵 **파란색**: 정보 메시지
- 🟢 **녹색**: 성공 메시지
- 🟡 **주황색**: 경고 메시지
- 🔴 **빨간색**: 오류 메시지

## 💡 사용 팁

### ✅ 효율적인 사용
1. **배치 처리 활용**: 여러 날짜 데이터를 한번에 처리
2. **로그 확인**: 오류 발생 시 로그에서 원인 파악
3. **옵션 조정**: 필요한 결과만 생성하여 시간 절약

### ⚠️ 주의사항
1. **폴더 구조**: L3A로 시작하는 올바른 폴더 구조 필요
2. **XML 파일**: `*_Aux.xml` 파일이 반드시 있어야 함
3. **디스크 공간**: 충분한 저장 공간 확보
4. **동시 실행**: 처리 중에는 다른 처리를 시작하지 마세요

### 🐛 문제 해결
1. **GUI가 실행되지 않음**
   - Python 설치 확인
   - tkinter 패키지 확인
   - `run_gui.bat` 사용

2. **처리 중 오류**
   - 로그 창에서 오류 메시지 확인
   - XML 파일 존재 여부 확인
   - GDAL 설치 확인

3. **느린 처리 속도**
   - 하드웨어 성능에 따라 다름
   - 배치 처리 시 시간이 더 소요됨
   - 불필요한 옵션 해제
