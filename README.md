# 국토정보위성 자동 처리 시스템

KOMPSAT L3A 레벨 위성 데이터 자동 지리보정 및 NDVI 계산 시스템

## 📋 개요

이 시스템은 국토정보위성(KOMPSAT) L3A 데이터를 자동으로 처리합니다:
- XML 메타데이터에서 좌표 정보 추출
- 4개 밴드(Blue, Green, Red, NIR) 지리보정
- RGB 합성 이미지 생성
- NDVI(정규식생지수) 자동 계산
- QGIS 프로젝트 생성 및 시각화

## 🎯 주요 기능

### 1. 자동 지리보정
- XML의 4개 코너 좌표를 GCP로 변환
- GDAL을 사용한 정확한 지리보정
- Korea2000 좌표계(EPSG:5179) 적용
- LZW 압축으로 파일 크기 최적화

### 2. NDVI 계산
- 공식: `NDVI = (NIR - Red) / (NIR + Red)`
- 범위: -1.0 ~ 1.0
- 해석:
  - `-1.0 ~ -0.1`: 물, 그림자, 나지, 도시
  - `0.0 ~ 0.2`: 식생이 거의 없음
  - `0.2 ~ 0.4`: 중간 정도의 식생
  - `0.4 ~ 0.8`: 건강한 식생
  - `0.8 ~ 1.0`: 매우 건강한 식생

### 3. QGIS 통합
- QGIS MCP 서버 연동
- 자동 레이어 로드
- 색상 램프 적용 (갈색-노랑-녹색)
- 프로젝트 파일(.qgz) 자동 저장

## 🔧 처리 단계

1. **XML 메타데이터 파싱**
   - Aux.xml 파일 읽기
   - 4개 코너 좌표 추출
   - 밴드 정보 확인

2. **지리보정 (Georeferencing)**
   - GCP(Ground Control Point) 생성
   - GDAL Translate로 VRT 생성
   - GDAL Warp로 좌표계 변환

3. **RGB 합성**
   - 3개 밴드(R, G, B) 병합
   - 16bit 멀티밴드 GeoTIFF 생성

4. **NDVI 계산**
   - NIR, Red 밴드 로드
   - NDVI 공식 적용
   - Float32 래스터 생성

5. **시각화**
   - QGIS 프로젝트 생성
   - 색상 램프 적용
   - 레이어 추가 및 저장

## 🔍 문제 해결

### GDAL 설치 오류
```bash
# Windows
pip install GDAL-3.x.x-cpXX-cpXX-win_amd64.whl

# Linux
sudo apt-get install python3-gdal
```

### 좌표계 오류
- XML 파일에 좌표 정보가 있는지 확인
- EPSG:5179 (Korea2000) 사용

### 메모리 부족
- 큰 이미지의 경우 타일 처리 사용
- COMPRESS=LZW 옵션으로 파일 크기 감소
